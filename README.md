# Chocotassium

A custom ui for Chocosploit (a private executor) that looks like Potassium

## Prerequisites

install or no work

- Node.js
- npm (comes with Node.js)
- have access to chocosploit

## Installation

1. Clone repo

2. install deps by doing `npm install`

## Buildin g

To create a binary:

```bash
npm run build
```

outputs in /dist

## Using

Download your "Injector.exe" from the Chocosploit server (and maybe your scripts folder) and drop them into where you downloaded Chocotassium, then run "Chocotassium.exe" or "npm start"
